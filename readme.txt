== Description ==

MOST_WANTED Wallet Discount Manager is a plugin that can be used to manage the usage of the wallet balance on the cart page.

It can be easily activated and configured, Tested for WordPress 5.7.2 and upper versions.

It comes with the following features.

- Type of using wallet balance (Percentage/Fix Amount).
- Deduct amount from total wallet balance or a specific amount from a specific product.

== Installation ==

1.Upload and/or install MOST_WANTED Wallet Discount Manager.
2.Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==

= Is it posible to deduct some percentage only form the total wallet amount? =

Yes, it is.

= Is it posible to deduct some percentage only for selected products? =

Yes, you can select specific products.
